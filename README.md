Набор лекций и задач для астрофизической школы "Траектория" https://astroschool.space/.

Полезные команды:

Установка uv (linux/macos):

```shell
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Установка uv (windows):
```shell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

Сихронизация зависимостей:
```
uv sync
```

Последняя версия репозитория:
```
git pull
```
